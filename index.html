Build Your Own Retro RPG Worlds

Design maps, craft adventures, and build 2D & 3D retro RPGs — with visual tools and Python scripting.

  • Design UI widgets and connect them to actions and events
  • Build top-down RPGs inspired by Ultima
  • Design immersive 3D worlds with classic vibes
  • Powerful event-driven logic with easy Python scripting
  • Craft procedural materials visually

Cross-Platform Freedom

Eldiron is built in Rust with zero hardware dependencies. Run the Creator and your games seamlessly across all platforms.

Multiple Game Styles

Create top-down 2D, isometric, or first-person 3D games—switch between them instantly, even during gameplay.

Powerful Inbuilt Editor

Design maps, build interfaces, or craft procedural materials using the intuitive Doom-style editor system.

Tile Everything Beautifully

Use your own tilesets or generate unique ones procedurally to texture floors, walls, and characters in your world.

Script with Python — Nodes Soon

Use the built-in Python editor to create behaviors, events, and logic. A node-based system is coming soon, built on the same Python modules for visual, flexible scripting.
